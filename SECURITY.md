# Security Policy

## Supported Versions

Security fixes are prioritized for the latest release line.

## Reporting a Vulnerability

Please report security issues privately and do not open public GitHub issues for vulnerabilities.

Send details to: `security@valuya.build`

Include:

- affected package(s)
- impact and attack scenario
- reproduction steps / PoC
- suggested mitigation (if available)

## Response Expectations

- Initial acknowledgment: within 3 business days
- Triage outcome: within 7 business days
- Fix timeline: depends on severity and exploitability

## Coordinated Disclosure

We support coordinated disclosure and will credit reporters who want attribution.
