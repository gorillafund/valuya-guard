# Agentokratia Guardian Template

Template for connecting a Guardian wallet signer to Valuya product purchase flow.

## Install

```bash
pnpm add @valuya/agentokratia-signer
```

## Required env

```bash
VALUYA_BASE=https://pay.gorilla.build
VALUYA_TENANT_TOKEN=ttok_...
```

## Run

Use this template by wiring real Guardian SDK methods in `index.ts`.
