## Documentation

## Community / Usage Questions

Open a GitHub Discussion or Issue with:

- framework/runtime
- package version
- request/response samples
- logs with sensitive values redacted

## Bug Reports

Please include:

- exact package(s) and versions
- minimal reproduction
- expected vs actual behavior
- environment details

## Security Reports

For security issues, use `SECURITY.md` reporting channel.
